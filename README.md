# 🚗 Fuel Cost Calculator

A modern Android application built using **Java** and **XML** that calculates the fuel required and total fuel cost for a single trip.

## ✨ Features

* ⛽ Calculate fuel required based on distance and mileage
* 💰 Calculate total fuel cost
* ✅ Input validation
* 🔄 Clear/Reset functionality
* 🎨 Material Design UI
* 🌙 Light and Dark theme support
* 📱 Responsive layout for different screen sizes

## 📐 Formula Used

Fuel Required = Distance ÷ Mileage

Total Fuel Cost = Fuel Required × Fuel Price
