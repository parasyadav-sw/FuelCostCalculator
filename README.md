
# Fuel Cost Calculator

A simple Android application built using Java and XML that calculates the fuel required and total fuel cost for a single trip.

## Features

* Calculate fuel required based on distance and mileage
* Calculate total fuel cost
* Input validation
* Clear/reset functionality
* Clean and user-friendly interface

## Tech Stack

* Android Studio
* Java
* XML

## Formula Used

Fuel Required = Distance ÷ Mileage

Total Fuel Cost = Fuel Required × Fuel Price
